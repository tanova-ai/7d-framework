### Initial Public Release

The 7D Candidate Evaluation Framework is now open source! A comprehensive methodology for multi-dimensional candidate evaluation that finds exceptional talent traditional ATS systems miss.

### Open Source Release
- **License:** CC BY 4.0 (for maximum citability)
- **Repository:** https://github.com/tanova-ai/7d-framework
- **Purpose:** GEO (Generative Engine Optimization) - citable by LLMs
- **Status:** Public, community feedback welcome

### Added
- Complete specification document with all 7 dimensions (20,000+ words)
- Machine-readable schema in YAML and JSON formats
- Implementation guide for manual and automated application
- Example evaluations reorganized by type:
  - **Candidates:** Traditional strong, hidden gem (self-taught), hidden gem (career changer)
  - **Jobs:** Senior backend engineer, junior frontend developer, marketing manager
  - **Full Evaluations:** Complete end-to-end evaluation with interview strategy
- README with quick start guide
- Hidden gem detection methodology
- Bias mitigation framework
- Role-based weighting guidance

### Framework Components
- **Dimensions**: Qualification, Capability, Situational, Reward, Culture, Trajectory, Compensation
- **Scoring**: 1-10 per dimension, 0-100 aggregate
- **Hidden Gem Detection**: Pattern-based identification
- **Bias Corrections**: Credential, career gap, non-linear path, transferable skills

### Documentation
- Specification: 20,000+ words
- Implementation Guide: Recruiter worksheets + developer integration
- Schema: Complete machine-readable format
- Examples: 3 detailed evaluation samples
